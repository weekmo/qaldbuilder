## To add maven dependency

## Add repository
```
<repositories>
  	<repository>
  		<name>Qald Builder</name>
  		<id>qaldBuilder</id>
  		<url>https://github.com/weekmo/qaldbuilder/raw/master/mvn-repo/</url>
  	</repository>
 </repositories>
```

## Add dependency
```
</dependency>
  	<dependency>
  		<groupId>org.hobbit</groupId>
  		<artifactId>qaldbuilder</artifactId>
  	<version>1.1.0</version>
  </dependency>
 </dependencies>
```
